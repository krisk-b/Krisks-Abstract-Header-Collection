# Krisk's Abstract Header Collection

### What is it?
Krisk's Abstract Header Collection provides lightweight, abstract tools for C and C++ projects. These files are **header-only** and do not require a `.c` counterpart to function. 

Additionally, this repository includes abstractions (IN THE FUTURE!) for other libraries such as:
*   **GLFW3**
*   **GLAD 2**
*   ...and more.

### How to use it?
Just drag and paste the header files into your C or C++ projects. 

> [!IMPORTANT]
> **Dependencies:** Some header files depend on each other. Each file includes a comment block at the top specifying its requirements.

### Documentation
Documentation is provided directly within the top comment block of each header file. It is designed to be straightforward for developers already familiar with the libraries being abstracted.
